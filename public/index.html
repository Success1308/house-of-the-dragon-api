Welcome to House of the Dragon API

Overview:

This API provides information about characters from House of the Dragon.

Get All Characters

Get a list of all characters.

Endpoint: /characters

Get Character by ID

Get information about a specific character by their ID.

Endpoint: /characters/{id} (replace `{id}` with a number)

Example Usage:

To get all characters:

GET /characters

To get character with ID 1:

GET /characters/1

Explore the API and have fun!
